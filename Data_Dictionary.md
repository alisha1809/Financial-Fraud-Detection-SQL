# Data Dictionary

| Column | Data Type | Description |
|---------|-----------|-------------|
| txn_id | INT | Unique transaction identifier |
| user_id | INT | Unique customer identifier |
| amount | DECIMAL(10,2) | Transaction amount |
| city | VARCHAR(50) | Transaction location |
| merchant | VARCHAR(100) | Merchant name |
| payment_method | VARCHAR(30) | Payment mode |
| txn_time | DATETIME | Date and time of transaction |
